Для большего удобства пользования "Background Copy" рекомендуется
установить макросы для вызова часто используемых команд плагина.
В папке Macros содержатся командные файлы с "говорящими" названиями,
каждый из которых устанавливает макрос для вызова одной команды плагина.

1. Присвойте плагину "горячую клавишу" - "b" (F4 в меню выбора плагинов).

2. Запустите все необходимые .cmd-файлы. Например, запуск BGmove.cmd
назначает вызов команды фонового перемещения файлов на клавишу CtrlShiftF6.
Чтобы назначить этот же макрос на другую клавишу, скажем, на CtrlAltF12,
запустите "BGmove.cmd CtrlAltF12" (без кавычек). Рекомендуется запускать
эти cmd-файлы из FAR - в этом случае будут учтены его языковые настройки и
текущий пользователь (если FAR запущен с ключом /u).

3. Перезапустите FAR.
