Класс FarFileInfo

Класс FarFileInfo представляет собой собрание статических методов, позволяющих получать различную информацию о файлах по их именам. Информация получается без открытия файлов.

Методы

static bool FileExists (const char *fileName)

Проверяет, существует ли файл с указанным именем.

static bool IsDirectory (const char *fileName)

Проверяет, является ли указанное имя файла каталогом. Если файла или каталога с таким именем не существует, также возвращает false.

static bool GetInfo (const char *fileName, WIN32_FIND_DATA &findData)

Заполняет структуру findData типа WIN32_FIND_DATA информацией о файле с указанным именем. Возвращает true при успешном завершении или false, если файла с таким именем не существует.

static __int64 GetFileSize (const char *fileName)

Возвращает размер файла с указанным именем, либо -1, если файла с таким именем не существует.
