
  !!! плагин тестировался в Far Manager 1.70 beta 5             !!!
  !!! работоспособность с другими версиями нижее гарантируется  !!!

  !!! для нормальной работы плагину нужна библиотека msvcrt.dll !!!

  устанавливать обычным способом:
   * создайте подподпапку с произвольным именем в папке %FARHOME%\Plugins;
   * скопируйте туда все, что находится в дистрибутиве плагина
   * перезапустите FAR

  как пользоваться:
   * установите курсор на файл сообщения или маилбокса и нажмите Ctrl+PgDn
   *

  плагины второго уровня служат для поддержки майлбоксов.

  в стандартную поставку входят плагины для просмотра след. маилбоксов:
   * unix-style (*.mbx)
   * thebat (определяется автоматически и может иметь любое имя)
   * outlook express 5.0 (*.dbx)
   * squish messagebase (*.sqd)
   * fido packet base (*.pkt)
   * folder (пзволяет просматривать папки с *.msg - смотрите example.fld)

  при просмотре маилбоксов можно жмякать:
   * F3      - просмотра текста сообщения в специальном диалоге
   * F4      - просмотра текста сообщения во встроенном редакторе
   * AltF3   - просмотра исходника сообщения во встроенном вьювере
   * AltF4   - просмотра исходника сообщения во встроенном редакторе
   * F5      - копирование сообщения/ний на другую панель.
   * CtrlF12 - изменение вида отображения тредов.

  плагин поддерживает префиксы: mv, mve, mvv и mb
  всем желательно передавать полное имя файла в качестве параметра.

  первый, вызывает диалог просмотра файлов электронных сообщений,
  второи, вызавает просмотр во встроенном редакторе,
  третий, во встроенном просмотрщике,
  последний, отображает содержимое майлбоксов, писем и ньюсов на файловой панели

  для настройки цветов плагина используйте меню конфигурации,
  настройка панели и других специфических параметров производится в ini-файле,
  надеюсь это временно :)

  для создания собственных файлов конфигурации для маилбоксов надо создать файл
  с тем же именем, что и маилбокс плюс (зависит от параметра SettingsExtension
  в секции Mailbox) ".mbs". например, конфиг для "messages.tbb"
  будет иметь имя "messages.tbb.mbs". дальше создаем  там  секцию  "Mailbox"  и
  используем любые параметры из головного конфига.
