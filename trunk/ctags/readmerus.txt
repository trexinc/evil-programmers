Скачать программу ctags.exe можно тут : http://ctags.sourceforge.net
Для С/С++ проектов запускать с такими ключами:
ctags --fields=+n --c++-types=+px --c-types=+px *
И -R для рекурсивного поиска.
Параметр --fields=+n опциональный, но несколько увеличит скорость поиска.

Файл tags можно загрузить из меню плагинов или из коммандной строки фара:
tag:tags

Для автозагрузки tags файлов нужно в корневой директории проекта
создать файл .tags-autoload в котором прописать
пути до tags файлов которые должны быть автоматически загружены
при вызове плагина из редактора для файлов проекта.
Так же можно в опциях плагина указать путь к файлу содержащему
имена tags файлов для автоматической загрузки при
вызове из любого файла.

Любой загруженный tags файл можно выгрузить из меню плагина.

Update можно производить только для загруженных tags файлов.
После update-а нужно еще раз перезагрузить tags файл,
прежде чем можно будет сделать еще один update.
